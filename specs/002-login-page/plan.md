# Implementation Plan: Login Page

**Branch**: `002-login-page` | **Date**: 2026-03-02 | **Spec**: [spec.md](spec.md)  
**Input**: Feature specification from `specs/002-login-page/spec.md`

## Summary

Redesign the login page and landing page for CausewayIncubator to match the reference design. Replace the current bare-bones Google OAuth login with a polished email-first experience featuring Firebase Email Link (passwordless) auth, anonymous "Continue as Guest" flow, and a branded landing page with hexagon decorations. The existing auth guard flow (consent → early-access → home) remains unchanged.

**Language/Version**: TypeScript ~5.4.x
